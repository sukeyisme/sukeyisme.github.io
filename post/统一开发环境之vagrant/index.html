统一开发环境之vagrant

 Posted on January 24, 2018

为什么选择vagrant

制作base box

之前用vagrant封装过开发环境到一半，去了docker阵营，docker更轻量，启动速度更快，就是带界面有点麻烦且不好用。 用x11-forwarding 可以在浏览器和PC，mac ，甚至手机，都可以操作界面。但是会发现在mac这种高清屏下x11支持有问题，很模糊 。vnc也不好用。所以又回到了vagrant，毕竟是虚拟机。

    vagrant package --base archlinux
    vagrant box add Arch package.box
    vagrant init Arch
    vagrant up

上传vagrant cloud
